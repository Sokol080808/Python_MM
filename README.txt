1) git clone https://github.com/Sokol080808/Python_MM.git - скачать репозиторий
2) cd Python_MM - зайти в репозиторий
3) git status - посмотреть статус измененных файлов
4) git add - приготовить файл к коммиту (сохранению)
5) git commit -m "описание" - "описание коммита", сохраняет подготовленные изменения на компьютере
6) git push origin main - загрузить сохраненные изменения на сайт сщьь
